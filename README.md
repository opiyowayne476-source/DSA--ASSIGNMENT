# Queue Implementation in C

# Overview
This project demonstrates a simple **queue data structure** implemented in C using a fixed-size array.  
It supports the following operations:
- **Enqueue**: Insert an element at the rear.
- **Dequeue**: Remove an element from the front.
- **Display**: Show all elements currently in the queue.

The program also handles **overflow** (queue full) and **underflow** (queue empty) conditions gracefully.

---

# Features
- Fixed-size queue with `MAX = 5`.
- Enqueue operation with boundary check.
- Dequeue operation with reset when empty.
- Display function to show current queue elements.
- Demonstrates both **overflow** and **underflow** cases.
